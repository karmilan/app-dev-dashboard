# App Dev Dashboard

A dashboard application.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/karmilan/app-dev-dashboard
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Technologies Used

- NextJs
- TailwindCSS
